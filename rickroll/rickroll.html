NVIDIA reveals GeForce RTX 50 series graphic cards release date. NVIDIA GeForce RTX 50 series reportedly features 28 Gbps GDDR7 memory and 512-bit bus. This is despite the fact that the first GDDR7 memory chips will be capable of 32 Gbps speeds. NVIDIA will also stick with 16 Gbit densities for the GDDR7 memory chips, which means memory sizes could remain largely unchanged for the next generation; with the 28 Gbps GDDR7 memory chips providing 55% higher bandwidth over 18 Gbps GDDR6 and 33% higher bandwidth than 21 Gbps GDDR6X.
Source: nvidia.com
